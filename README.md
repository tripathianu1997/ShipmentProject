## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev

In this Project You can see Three Faces Driver , User & Admin 
Each faces have own Dashboard 
login Authentication Use NextAuth And session control use by NextAuth 
User registration ,Driver Registration .
user create shipment and track Shipment Status and admin also create multiple user and shipment and driver.
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
